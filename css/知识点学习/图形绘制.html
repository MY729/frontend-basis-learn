# 2.利用伪元素绘制两个圆形

两个圆形的圆心分别在正方形的上边和右边中点

.heart:before {
  position: absolute;
  top: -50px;
  content: "";
  width: 100px;
  height: 100px;
  border-radius: 50%;
  background: lightgreen;
}
.heart:after {
  position: absolute;
  left: 50px;
  content: "";
  width: 100px;
  height: 100px;
  border-radius: 50%;
